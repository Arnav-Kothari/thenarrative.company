# The Narrative Company - Project Instructions

## Design System

When building any UI, components, or pages for this project, use shadcn/ui as the default component library. Always invoke `/shadcn` before building UI components to pull the latest docs and patterns. ALWAYS read `brand.html` in this directory first and style all components according to The Narrative Company brand guidelines exactly. This includes colors, typography, spacing, components, and voice & tone.
